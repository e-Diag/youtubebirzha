Политика конфиденциальности

Последнее обновление: 2024

1. Сбор информации

Мы собираем следующую информацию:

  • Telegram User ID (для идентификации пользователя)
  • Username (если указан пользователем)
  • Информация из объявлений, размещённых пользователем
  • IP-адрес (для обеспечения безопасности)

2. Использование информации

Собранная информация используется для:

  • Предоставления функционала сервиса
  • Обеспечения безопасности
  • Модерации контента
  • Предотвращения мошенничества

3. Защита данных

Мы принимаем меры для защиты ваших данных:

  • Использование шифрования при передаче данных
  • Ограничение доступа к персональным данным
  • Регулярное обновление систем безопасности

4. Логирование

Мы логируем только следующую информацию:

  • User ID (без других персональных данных)
  • Путь запроса и метод
  • IP-адрес
  • Время запроса

Мы НЕ логируем:

  • Имена пользователей
  • Содержание объявлений
  • Другие персональные данные

5. Передача данных третьим лицам

Мы не передаём ваши персональные данные третьим лицам, за исключением случаев, предусмотренных законодательством.

6. Права пользователей

Вы имеете право:

  • Запросить информацию о ваших данных
  • Запросить удаление ваших данных
  • Отозвать согласие на обработку данных

7. Cookies и аналогичные технологии

Мы не используем cookies для отслеживания пользователей.

8. Изменения в политике

Мы оставляем за собой право изменять данную политику конфиденциальности. Изменения вступают в силу с момента публикации.

9. Контакты

По вопросам конфиденциальности обращайтесь: @birzha_manager
